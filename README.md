# Loan Prediction Monitoring System

Proyek ini adalah implementasi end-to-end Machine Learning Operations (MLOps) untuk prediksi pinjaman. Menggunakan Flask sebagai API, Docker untuk containerization, dan Prometheus untuk monitoring.

## Teknologi yang Digunakan
- **Python** (Pandas, Scikit-Learn, Flask)
- **Docker** & **Docker Hub**
- **Prometheus** (Monitoring)
- **GitHub Actions** (CI/CD)
